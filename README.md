# zoltraak

## Local Development

Prerequisites:

- Node.js 20 or newer
- npm

Install dependencies:

```sh
npm install
```

Start the local dev server:

```sh
npm run dev
```

Vite prints the local URL in the terminal, usually `http://localhost:5173/`.

Run the production build:

```sh
npm run build
```

Run the end-to-end tests:

```sh
npm run test:e2e
```
